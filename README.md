# Bookmark Manager

A simple web application that allows users to save and display their favorite websites as bookmarks.

---

## **Demo**

You can enter the website name and URL, and it will be displayed in a list. Each URL is clickable and opens in a new tab.

---

## **Features**

- Add new bookmarks with a site name and URL.
- Basic input validation:
  - Site Name cannot be empty.
  - Site URL cannot be empty.
- Display bookmarks dynamically in a list.
- Clickable URLs open in a new browser tab.
- Error messages displayed for invalid inputs.

---

## **Technologies Used**

- HTML
- CSS
- JavaScript (Vanilla JS)

---

## **How to Use**

1. Open `index.html` in your browser.
2. Enter the **Site Name** and **Site URL** in the input fields.
3. Click **Submit**.
4. The bookmark will appear in the list below the form.
5. Ensure the URL starts with `https://` or `http://` for proper navigation.
